# IndustryX Footer Credit Design

Add a `Brought to you by IndustryX` credit to the shared footer. `IndustryX` links to Malcolm Khong's GitHub profile in a new tab. Its size remains fixed while its primary color and text glow gently strengthen and soften, creating a grow-like effect without layout movement. Hover and keyboard focus make the link brighter immediately. Reduced-motion users receive the same stable color and hover treatment without the repeating animation.

The external URL belongs in shared site configuration. The footer renders the configured label and URL, while the global stylesheet owns the reusable glow animation.
